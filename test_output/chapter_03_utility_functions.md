# Chapter 3: Utility Functions

This chapter explains the utility functions that support the calculator's operations, including number formatting, validation, and basic statistical computations. These are reusable helpers that enhance the application's functionality without being tied to specific calculations.


## Related Files

- `utils.py`


## Sections

### Number Formatting

format_number takes a float and formats it to a specified number of decimal places, useful for displaying results cleanly.


### Validation

validate_positive checks if a value is greater than zero, which could be used for inputs like radii.


### List Operations

sum_list and average provide basic aggregation functions for lists of floats, demonstrating type hints and handling empty lists.
